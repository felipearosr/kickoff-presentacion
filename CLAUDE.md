# Kickoff Presentacion - Slidev Project

## Overview
Project kickoff presentation template built with Slidev. Corporate-style, Spanish language.

## Tech Stack
- **Slidev** — Markdown-based presentation framework (Vue-powered)
- **Mermaid** — Diagrams (Gantt charts for cronograma, flowcharts for architecture)
- **UnoCSS** — Utility-first CSS (built into Slidev)

## Commands
- `npm run dev` — Start dev server with hot reload
- `npm run build` — Build static SPA
- `npm run export` — Export to PDF

## Project Structure
- `slides.md` — Main presentation file (all slides)
- `styles/` — Custom CSS overrides
- `components/` — Custom Vue components (if needed)
- `public/` — Static assets (images, logos)

## Images
- Source images from **Pexels** (https://pexels.com) — free for commercial use, no attribution required
- Download via direct URL: `https://images.pexels.com/photos/{ID}/pexels-photo-{ID}.jpeg?auto=compress&cs=tinysrgb&w=800`

## Conventions
- All slide content in Spanish
- Animations should be subtle and professional (v-click, fade transitions)
- Use Mermaid for diagrams (Gantt, flowcharts) — avoid static images when possible
- Color palette: white background, stone/rose accents (bg-stone-50, border-stone-200, text-rose-600, text-stone-700)
- No emojis — use lucide icons if icons are needed
- Font sizes: titles 2xl+, body text base-lg
